福井大学附属国際原子力工学研究所原子炉熱水力部門　渡辺研究室のホームページです。

当研究室では，原子炉内の熱流動に関する解析的・実験的研究を行っています。

解析ではBest Estimateコードと呼ばれる，原子炉を1次元の流路で模擬したコード及び解析対象を詳細なメッシュで分割して解析を行う数値流体力学（CFD）コードを用いて研究を進めています。
