数値流体力学（CFD）

CFDを用いた研究

本研究室ではCFD (Computational Fluid Dynamics 数値流体力学）のコードを用いて、様々な流動の研究を実施しています。

レベルセット法を用いた浮遊液滴の数値解析

近年、液滴を静電場や超音波によって浮遊させる浮遊 液滴技術を、高融点材料の非接触熱物性測定や、分析 化学におけるサンプルの前処理等へ応用する事が期待 されています。しかし、液滴界面の大変形や液滴内外の 複雑な流動場形成などの非線形挙動が現れるため、技 術の確立には更なる研究が要されています。

本研究室では、Level-set法を用いた二相流数値解析に よって、浮遊液滴に見られる複雑な挙動を明らかにする事を目的としています。

PTSの数値

原子炉において何らかの事故が発生し、高温高圧条件下にある圧力容器に非常用冷却水が注水されることで 発生する熱衝撃をPTS　(Pressurised Thermal Shock)と言い ます。PTSによって圧力容器壁面上のクラックに引張応力 が作用した時、圧力容器が破損する事が懸念されます。

本研究室では、健全性評価のための構造解析に必要 な温度分布を、DESと呼ばれる乱流モデルを用いた熱水力数値解析をによって、比較的高精度・低コストで求める 事を目的としています。
